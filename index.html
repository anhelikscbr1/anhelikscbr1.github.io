Face Recognition App with FaceNet and MTCNN

The main objective of this project was to create a Graphical User Interface in Python that receives a photograph of a person given by the user and find the best face match of a database of persons. The program uses FaceNet and MTCNN technologies for the creation of an embedding for each face, and a vector database to find the closer coincidence.
